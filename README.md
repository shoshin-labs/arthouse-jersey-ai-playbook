# ArtHouse Jersey AI Playbook

A polished static microsite explaining how ArtHouse Jersey could implement AI agents and helpers across the charity, with a clear section for where consultants will help.

## Local preview

```bash
./serve.sh
```

The script auto-picks a free local port starting at 8093 and prints the URL.

## Build notes

- Static HTML/CSS/JS only
- No external images required
- GitHub Pages ready from the repo root
